.. toctree::

The purpose of this lab is to familiarize the Student with Per Request Policies

Objective:

-  Gain an understanding of Per Request policies

-  Gain an understanding of Step-up Authentication

-  Gain an understanding of Group Control

An APM Per-request policy runs when a client initiates a session.  Depending
on the actions included in the access policy, it can authenticate a user
and perform group or class queries to populate session variables with data
for use throughout a session.
